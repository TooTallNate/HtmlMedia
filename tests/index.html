Clicking on any of the <audio> or <video> nodes below will redirect to the test page using the specified HTML source. If the browser can play the media with it's native HTML5 media implementation, then that will be used, otherwise, HtmlMedia will kick in and provide playback support through Flash.

This is essentially as simple as it gets, a barebones <audio> example playing an MP3: 

<audio src="sounds/thanks.mp3"></audio>

And a barebones <video> example playing an H.264 encoded video: 

<video src="videos/chihuahua.mp4"></video>

With HtmlMedia, you can easily embed Flash encoded FLV files (better known as the On2 VP6 codec) using nothing but standards-compliant HTML5 markup. 

<video src="videos/chihuahua.flv"></video>
